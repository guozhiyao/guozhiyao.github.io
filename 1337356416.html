leetcode_1031：两个非重叠子数组的最大和

发表于 2019-05-05 | 分类于 algorithm ， LeetCode | 评论数： | 阅读次数：
本文字数： 2.5k | 阅读时长 ≈ 2 分钟

题目：
给出非负整数数组 A ，返回两个非重叠（连续）子数组中元素的最大和，子数组的长度分别为 L 和 M。（这里需要澄清的是，长为 L 的子数组可以出现在长为 M 的子数组之前或之后。）

从形式上看，返回最大的 V，而 V = (A[i] + A[i+1] + … + A[i+L-1]) + (A[j] + A[j+1] + … + A[j+M-1]) 并满足下列条件之一：
0 <= i < i + L - 1 < j < j + M - 1 < A.length, 或
0 <= j < j + M - 1 < i < i + L - 1 < A.length.

思路：
首先是最简单的暴力搜索方法，遍历两遍数组，把不重叠的最大值找出来，超时了。

其次对其进行简单优化，先分别遍历一遍计算出长度为L和M的子串的大小，然后再像上面的方法找到答案，这里解决了冗余求sum的过程。

304(59.18%)/12MB(100.00%)，结果看起来还行，但是还能再进一步优化。
对于求到的某个子串，比如A[i:i+L]，与其对应的应该是不重叠的长度为M的子串的最大值，这个应该是固定的，不需要重复遍历，可以用二维DP，dp[i][0]表示在i处，长度为M的子串的最大值，而dp[i][1]表示在i处，长度为L的子串的最大值。而答案ans的更新是比较：①之前的ans；②包含当前数字的前L个字符串以及之前的最长M的值；③包含当前数字的前M个字符串以及之前的最长L的值。

72ms(92.68%)/13.1MB(100.00%)
