使用V2Ray实现科学上网。

#前言
之前使用shadowsocket搭的梯子崩了，后面一直尝试都搭不起来，现在换成V2Ray就成功了。以下介绍一下搭建过程，主要分成服务器搭建和客户端搭建。

服务器端

这里服务器是从vultr 上购买的，以下分成两部分，购买服务器和配置服务器。

从vultr购买服务器

  1. 首先在vultr网站上注册一个账号。
  2. 点击左侧的“Servers”，然后点击左侧的加号，部署新的服务器
  3. 进入后，推荐使用亚洲的服务器，比如日本或者新加坡，距离比较近，延时比较小。然后使用Debian9操作系统，然后部署服务器。
  4. 等待一段时间后就能看到在“Servers”下多了一个服务器

    点击进去，可以看到服务器的IP地址、用户名以及密码，这些后面需要。
  5. 还需要测试该服务器的22端口是否开启，如果关闭的话就无法使用了。可以使用这个网址 测试，将服务器IP地址和端口输入，就能知道该端口是否开启
  6. 如果服务器的22号端口是关闭的，就需要重新部署新的服务器，直到找到22号端口是开启的服务器。这里vultr是计时收费的，所以不会有额外费用。

配置服务器，安装V2Ray

然后要对服务器进行配置，安装V2Ray

  1. 我这里使用putty远程登录服务器，使用刚刚得到的IP地址、用户名和账号可以登录到服务器上
  2. 然后使用一键安装脚本
    bash <(curl -s -L https://git.io/v2ray.sh)
    输入后，可以看到以下界面，就是开始安装V2Ray
  3. 按照命令行提示，选择‘安装’——‘TCP’，然后自定义V2Ray端口号，我这里设置为10086；不开启“广告拦截”；建议还是开启ss，然后设置ss的端口号注意不要和V2Ray冲突，然后输入ss的登录密码，选择默认的加密协议，这样就搭建完成了。
  4. 然后输入
    v2ray url
    可以获取服务器的vmess，后面需要。

客户端

然后需要在自己电脑上配置客户端

  1. 从V2RayN上下载最新的V2RayN，我这里使用的版本是2.26版本的，这个是V2Ray的GUI。
  2. 点击V2RayN的“检查更新”中的“检查更新v2rayCore”，会自动同github上下载核心代码。
  3. 复制刚刚服务器上的vmess，然后在安装好的V2RayN上，点击左上角的“服务器”中的“从剪切板导入批量URL”，就能看到下面添加了一项服务器信息，那个就是你购买的服务器。
  4. 然后点击该服务器，按回车，就能开启这个服务器。可以右键点击“测试服务器延时”，，如果出现*ms，说明服务器正常。
  5. 右键点击任务栏中的V2RayN，选择“启动Http代理”，然后选择“Http代理模式”中的”PAC模式”。
    PAC模式和全局模式的区别：PAC模式会维护一张PAC，只有境外的网址才会使用代理，而全局模式下，访问的所有网址都使用代理，这里建议使用PAC模式。

至此科学上网工具就搭建完成了。如果还是不能正常的科学上网，还需对浏览器进行配置。

浏览器

Chormone

  1. 需要先下载扩展工具“SwitchyOmega”
  2. 点击下载好的SwitchyOmega，选择“导入”，并且在“在线恢复”中输入‘https://github.com/FelisCatus/SwitchyOmega/wiki/GFWList.bak’，并点击“恢复”
  3. 就会出现一个GFWed的栏目，这里其实就是恢复了一张GFWList，它通过DNS地址进行区分，在Gfwlist列表的DNS走国外，其他走国内，该list包含所有需要走国外的DNS域名。
  4. 点击“GFWed”后，设置“代理端口”和V2RayN的“本地监听端口”相同。
    V2RayN的“本地监听端口”在“参数设置”中的“基础设置”中

至此，就能使用Chrome进行科学上网了。

IOS

如果还需要在IOS上进行科学上网，还需要以下步骤

  1. 从这里 下载shadowrocket，得到ipa
  2. 将手机连接电脑，使用PP助手从本地安装APP，就能得到shadowrocket，开启后是以下界面
  3. 点击右上角的加号，配置新的服务器。

    如上图所示，“类型”选择“Vmess”，“服务器”为你购买的服务器的IP地址，“端口”为你设置的V2Ray的端口，我这里为10086，“UUID”可以通过在V2RayN中双击你构建的服务中获得。

至此，就能够在IOS上进行科学上网了。
