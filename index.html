个人服务器+宝塔+nextCloud——构建个人云

发表于 2019-05-06 | 分类于 其他 | 评论数： | 阅读次数：
本文字数： 1.6k | 阅读时长 ≈ 1 分钟

使用个人服务器、宝塔以及nextCloud构建个人云存储。这里主要介绍使用宝塔和nextCloud，个人服务器的搭建可以参考之前的blog

购买和配置域名

之前的教程搭建的服务器，只能通过tqdm或者其他远程登录工具登录到服务器上，这里要构建成云存储服务器，最好给服务器购买一个域名。我这里是在阿里云上购买的，以下是购买域名的简单步骤。

  1. 首先到阿里云 上注册一个账号。
  2. 在上方搜索栏搜索“域名”，点击“域名注册”
  3. 在以下界面查询并购买域名
  4. 完成实名制认证和邮箱认证，大概需要花两天的事件。
  5. 点击右上角的“控制台”，进入以下界面
  6. 点击左侧”域名”进入以下界面，点击自己域名的解析
  7. 然后按照下面配置域名

安装宝塔面板

宝塔面板 是一个简单的Linux/Windows面板，能够一键创建网站、FTP、数据库；安全管理，文件管理，PHP多版本共存及切换；ASP+ASPX+PHP+MYSQL+MSSQL05/08+mariaDB+FTP、一键SSL，你能想到的，它都能做到！不需要多余的操作，只需安装一个宝塔面板，就能一键配置好服务器环境！
接下来在服务器上安装宝塔面板

  1. 安装宝塔面板

  2. 等几分钟后，如果安装成功，就能看到以下界面

    注意保存图像中的那部分信息，它是用来登录宝塔面板的。

  3. 关闭安全登录入口
    rm -f /www/server/panel/data/admin_path.pl
  4. 在浏览器中打开域名:8888，并使用上面的账号密码登录
  5. 进入后安装LNMP
  6. 也可以修改登录账号密码等等。

NextCloud

Nextcloud 是一个免费专业的私有云存储网盘「开源」项目，可以让你简单快速地在个人/公司电脑、服务器甚至是树莓派等设备上架设一套属于自己或团队专属的云同步网盘，从而实现跨平台跨设备文件同步、共享、版本控制、团队协作等功能。

安装NextCloud

以下介绍安装NextCloud的方法

  1. 进入宝塔面板，按照以下步骤输入
    https://download.nextcloud.com/server/releases/nextcloud-15.0.2.zip
    安装NextCloud
  2. 然后直接将下载的压缩包点击“解压”
  3. 点击“网站”——“添加站点”，然后按照下面的进行配置
  4. 配置完成会，会出现“数据库账号资料”，注意，先不要关闭，后面需要用到
